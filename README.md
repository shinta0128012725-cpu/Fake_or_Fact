# Fake News 判定モデル（Fake or Fact Classification）

## 概要

本プロジェクトでは、ニュース記事のテキストデータをもとに、情報が **フェイクニュース（Fake）** か **事実（Fact）** かを判定するテキスト分類モデルを構築しました。

自然言語処理（NLP）を用いて文章の特徴を数値化し、機械学習モデルによる二値分類を行うことで、誤情報検知の基礎的な仕組みを実装しています。

---

## 使用技術

* Python
* Pandas / NumPy
* scikit-learn
* Natural Language Processing（NLP）
* Matplotlib / Seaborn
* Jupyter Notebook

---

## データ前処理

### 1. テキストクリーニング

* 小文字化
* 記号・不要文字の除去
* ストップワードの除去

### 2. ベクトル化

テキストデータを機械学習モデルで扱えるよう数値化。

例：

* Bag-of-Words
* TF-IDF

```id="tfidf_flow"
テキスト → TF-IDFベクトル
```

---

## モデル構成

* 問題設定：二値分類（Fake / Fact）
* 使用モデル：

  * ロジスティック回帰
  * または分類アルゴリズム（※Notebook構成に応じて）

設定：

* 学習データとテストデータに分割
* Accuracyなどの指標で評価

---

## 学習と評価

* 学習データでモデルを訓練
* テストデータで性能評価
* 評価指標：

  * Accuracy
  * 混同行列（Confusion Matrix）

フェイクニュース判定タスクにおいて一定の分類性能を確認。

## まとめ

本プロジェクトでは、ニューステキストを対象にフェイクニュース判定モデルを構築し、テキスト前処理から特徴量設計、モデル学習、評価までの一連のNLPパイプラインを実装しました。
